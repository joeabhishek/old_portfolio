Carve.js

Carve is a JavaScript library to process large chunks of data in the browser. It lets you create a datasource in JavaScript and apply select, filter, sort, aggregate, group and limit on the data. It can slice through more than 50,000 records very easily. A Carve datasource can also be extended and cloned.

But why?

For almost every data visualization or data analysis project that I've worked on, I have found myself re-implementing a lot of data processing routines. As such, I felt the need to develop a JavaScript library which can not only process the data efficiently but also let the user run a wide range of queries on the data. This library comes very handy when visualizing data from very large datasets and when applying queries on the fly is important.

Usage

First create a new Carve datasource. You have to specify the fields and their types. Currently, 'string' and 'num' are supported. DataURL is the path to your data in JSON format. 

    var ds = new Carve.DataSource({
        fields: {
                'firstName': {
                        'type': 'string'
                },
                'age': {
                        'type': 'num'
                },
                'param1': {
                        'type': 'num'
                },
                'param2': {
                        'type': 'num'
                }
        },
        dataURL: "/path/to/data.json"
    });

You can then apply required operations on the datasource. For instance,

    var querySet = ds.query()
        .select([
          {field: 'age', aggregate: true, aggregateFunc: 'sum' }, 
          {field: 'firstName', groupBy: true}
        ])
        .sort({field: 'age', order: 'desc'})
        .filter({field: 'age', operator: '>', value: 32})
        .group([{field: 'firstName'}])
        .limit({skip: 3, take: 10})
        .asArray(function(data){
          console.log(data);
        });

asArray returns the processed data to a specified callback function.
