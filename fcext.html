FusionCharts ExtJS extension

FusionCharts is an exhaustive charting library. Using FusionCharts with ExtJS becomes extremely cumbersome. This extension lets you use FusionCharts charts as native ExtJS component. You can use FusionChats ExtJS component in either ExtJS MVC way or as a normal ExtJS application. You can download two fully working examples that are included along with the source. This extension uses a trial version of FusionCharts.

Basic Usage

Require the extension before using the FusionCharts ExtJS component

    Ext.require(['FusionCharts.core.Chart'])

Use the FusionCharts component inside your application like any other native ExtJS component

    Ext.create('Ext.container.Viewport', {
        layout: 'fit',
        items: [
            {
               xtype: 'fusioncharts',
               title: 'JSON URL',
               chartType: 'Column3d',
               chartID: 'firstChart',
               chartWidth: '400',
               chartHeight: '400',
               debugMode: '0',
               JSONUrl: 'data/chartData.json',
            }
        ]
    })
